# **App Name**: Kkom-Morning Companion

## Core Features:

- Login Authentication: Authenticate users via a 4-digit PIN, stored locally.
- Dashboard Display: Display personalized greetings, current date, weather, air quality, and outfit recommendations on a central dashboard.
- Weather Data Fetching: Fetch and display weather data, including condition, temperature, feels-like, and high/low ranges (using mock data).
- Air Quality Monitoring: Fetch and display air quality data, including grade, PM10/PM2.5 values, and health messages (using mock data).
- Outfit Recommendation: Recommend outfits based on temperature and air quality conditions, including a mask warning when air quality is poor.
- Outfit Generation: Use a generative AI tool to find trending outfits based on the current weather and air quality.

## Style Guidelines:

- Primary color: A soft mint green (#96e6a1), inspired by the gradient background.
- Background color: Light gray (#f9fafb) for a clean, mobile-friendly appearance.
- Accent color: A gentle lime green (#d4fc79), providing a refreshing and harmonious feel with the primary mint green.
- Headline font: 'Poppins', sans-serif, for bold headings and short body text.
- Body font: 'PT Sans', sans-serif, paired with Poppins for a balanced feel between modernity and readability.
- Utilize Lucide React icons for a consistent and modern aesthetic across the application.
- Mobile-first, center-aligned layout with max-width 640px for optimal viewing on smaller screens.
- Subtle shake animation on invalid PIN code entry for immediate user feedback.
